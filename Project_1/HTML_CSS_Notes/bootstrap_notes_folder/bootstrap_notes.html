#1 - What's Bootstrap?


Bootstrap is a front-end design framework (library) for web developers. Allows your website to
be responsive (adaptable layout) to different view-ports.

Reference: Bootstrap WebSite

#8 - Code Refactoring Principles


  1. Readability - Easy to understand, logically organized, and commented
  2. Modularity (re-use code or ease to narrow down, easier to maintain)
  3. Efficiency - How fast does your code run?
  4. Length - Just make sure it's readable and comprehensible
