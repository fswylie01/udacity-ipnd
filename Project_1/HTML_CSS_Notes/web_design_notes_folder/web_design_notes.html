#1 - Designer vs. Non-Designer Thinking


Think about colors (color theory), and typography with different typefaces.


#2 - Colour Theory


Different colors or different shades can achieve different results.

  • Red - Love, Energy, Intensity (used in car ads)
  • Yellow - Joy, Intellect, Attention (used in grab attention - beach body ads)
  • Green - Freshness, Safety, Growth (used in food service ads)
  • Blue - Stability, Trust, Serenity (used in medical and health ads)
  • Purple - Royalty, Wealth, Femininity (used in loan ads)

#3 - Combine Colours to Create Colour Palettes


Start with the color wheel.

  • Analogous Colors Palette or adjacent color - used for app background screens
  • Complementary Color Palette or opposite colors - used for attention grabbing screen shots
  • Split Color Palette to tone down the overall color - used for simple app icon design
  • Triadic Color Palette are well balanced and attention grabbing - look very 90s designs
  • Monochromatic Palette is for modern design

#5 - Introduction to Typography & Readability


Serif Type Family - Have tiny little feet dating back to stone carvings.

Divided into four font families:

  1. Old Style (1400s style)
  2. Transitional (new times roman font)
  3. Modern (vogue magazine)
  4. Slab-Serif (newspaper print)

Sans-Serif Type Family - Does NOT have those tiny little feet.

  1. Grotesque (news gothic font)
  2. Neo-Grotesque (helvetica)
  3. Humanist (gill sans)
  4. Geometric (futura)

Typography Determines Readabilty - Look for the following:

  1. Open Shapes - larger letters are easier to read
  2. Ample Intercharacter Spacing - more distance between letters are easier to read
  3. Unambiguous Forms - letters and numbers are easier to notice
  4. Varying Proportions - different character widths enhance readability

#6 - Combining Fonts


Rule: 2 fonts together will usually look pretty good, but 3 plus will not.

  • Things to keep Similar: Mood & Time Era of the fonts
  • Things to keep in Contrast: Serif-ness & Weights

#7 - UI Design


Start by imagining your APP as a journey any your the tour guide.

  1. Journey should be curated and tailored, and not a block text design.
  2. The importance of alignment and how it's key in digital design.
    Try to minimize the number of alignment lines.
  3. Make buttons appear clickable - ie. rounded border, add color gradients, & instruction
    image on button. Make it look interactive.
  4. Start you design working in monochrome and get alignment correct before adding color.
  5. Make text visible on your images - use a dark background, change text color for
    contrast, or change the gradient opacity across image.
  6. Be an Attention Wizard - construct a user journey to keep their attention.
